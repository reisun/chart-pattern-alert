# Support→Resistance Flip（切り上げレジサポ転換・下方向）

## 概要

上向きの支持線を**下に割れて**、その線が今度は**抵抗線になる**パターン。

```
          ／  ← 支持線 (上向き、複数タッチ)
         ／
      ＊／    ＊＊
       /    /＼＼ ＼＼＼   ＊
      / ＊／    ＼＼＼＼／
     ／       ↑ブレイク  ＼  リテストで跳ね返される
   ＊／                    ＼
    ＊                       ＊
```

## 意味合い

- それまでの上昇基調が崩れる
- 支えだった線が逆に重しになる

## 見るポイント

- ただ割れるだけでなく、割れたラインが**抵抗として機能**するかを確認
- リテスト時に跳ね返されるのが確認できると信頼度 UP

## 検出アルゴリズム

1. 直近のトレンドライン候補として、**上向きの安値列**に線形回帰（正の傾き、タッチ 2 回以上）
2. ある足 `i` で `close` が支持線を下抜け
3. 抜け後の N 本で高値が線近辺に触れ、かつ超えないまま反落
4. 3 を満たした時点で確定

## 閾値

- Resistance→Support Flip と共通設定（方向のみ反転）

## マーカー

- ブレイク足に `arrowDown` 仮、リテスト成功で塗りつぶし、略号 `Flip↓`
