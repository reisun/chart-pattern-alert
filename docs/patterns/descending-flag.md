# Descending Flag（下降フラッグ）

## 概要

強い下落のあとに少し戻して揉み合い、再び下落する。

```
\
 \
  \ 推進波（強い下落）
   \
    \___←─＼＼＼ （揉み合い: 平行チャネル、やや戻し）
        ＼＼＼＊
            ↓ブレイク
             ＼
              ＼
```

## 意味合い

- 急落後の自律反発に見えるが、下落トレンド継続

## 見るポイント

- 直前の**強い下落**（pole）
- 戻しは短く小さい
- **下抜け**で継続

## 検出アルゴリズム

1. 直近で下方向の推進波を特定（`flagPoleMinPct` 以上の下落）
2. pole 終点以降の 5〜25 本で高値/安値の回帰線、`flagSlopeMax` 以下かつ平行
3. チャネル**下辺を下抜け**（break 足の `close` < 下辺）で確定

## 閾値

- Ascending Flag と同設定、方向のみ反転

## マーカー

- ブレイク足に `arrowDown`、略号 `D-Flag`
