# Descending Triangle（下降トライアングル）

## 概要

**下値水平 / 上値切り下がる** 三角形。

```
＊─＊ 高値は切り下げ
  \ \
   \ \
    \ ＊─＊
     \   \
      ＊──＊──＊──＊  ← 下辺: 水平サポート（複数タッチ）
                   ＊← ブレイク（下抜け）
```

## 意味合い

- 下では買われるが戻り高値はどんどん低い
- 売り圧力が強まっている

## 見るポイント

- 高値が段々低くなる
- 何度か下値にぶつかった後、割れやすい

## 検出アルゴリズム

1. Pivot 低値列に線形回帰 → 水平付近（`|slope|` <= `triHorizTol`）
2. Pivot 高値列に線形回帰 → 負の傾き（`slope < 0`）
3. タッチ回数条件
4. 直近 `close` が下辺を下抜け → 確定

## 閾値

- Ascending Triangle と共通（方向のみ反転）

## マーカー

- ブレイク足に `arrowDown`、略号 `D-Tri`
