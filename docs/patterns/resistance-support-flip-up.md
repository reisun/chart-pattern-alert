# Resistance→Support Flip（切り下げレジサポ転換・上方向）

## 概要

下向きの抵抗線を**上に抜け**、そのラインが今度は**支持線として機能**する形。

```
 ＊\                ← 抵抗線 (下向き、複数タッチ)
    \
＊    \                    ＊ ← 抜け後に支持線として支えられる
 \    \             ＊    /
  \    \＊         / \  /
   ＊    \＊＊＊＊＊   ＊＊   ← ブレイク
           ＼＼＼＼←ライン抜け
```

## 意味合い

- それまでの下落基調が崩れる
- 「上値抵抗だった線」を超えたことで地合いが変わる

## 見るポイント

- ただ抜けるだけでなく、抜けた後に**そのライン付近で支えられる**（リテスト成功）
- これが確認できると信頼度が上がる

## 検出アルゴリズム

1. 直近のトレンドライン候補として、**下向きの高値列**に線形回帰（負の傾き、タッチ 2 回以上）
2. ある足 `i` で `close` が抵抗線を上抜け
3. 抜け後の N 本で安値が線近辺（`|low - line(t)| / line(t) <= lineRetestTol`）に触れ、かつ割り込まない
4. 3 を満たした時点で確定（リテスト成功）

## 閾値

| 名称             | 既定  |
|------------------|-------|
| `lineTouchMin`   | 2     |
| `lineRetestTol`  | 0.3%  |
| `retestMaxBars`  | 10    |

## マーカー

- ブレイク足に `arrowUp` 仮、リテスト成功で塗りつぶし、略号 `Flip↑`

## 実装メモ

- ラインを延長して将来のバーに投影する必要あり（lightweight-charts は line-series で表示可）
- リテスト判定は仮検出→確定までに猶予を持たせる
