# Ascending Triangle（上昇トライアングル）

## 概要

**上値ほぼ水平 / 下値切り上がる** 三角形。

```
───＊─＊───＊───＊──  ← 上辺: 水平レジスタンス（複数タッチ）
      \    /   \  /
       \  /     \/＊← ブレイク（上抜け）
        \/
         ＊─＊─＊
            下値は切り上げ（右肩上がり）
```

## 意味合い

- 上は売られるが、下から買いが入り続ける
- 買い圧力が強まっている

## 見るポイント

- 安値が段々高くなる
- 何度か上値にぶつかった後、突破しやすい

## 検出アルゴリズム

1. 直近の Pivot の高値列に線形回帰 → 傾きが水平付近（`|slope|` <= `triHorizTol`）
2. 安値列に線形回帰 → 正の傾き（`slope > 0`）かつ `|slope|` > `triSlopeMin`
3. 高値タッチ回数・安値タッチ回数それぞれ 2 回以上
4. 直近 `close` が上辺を上抜け → 確定

## 閾値

| 名称            | 既定  |
|-----------------|-------|
| `triHorizTol`   | 0.1%/本 |
| `triSlopeMin`   | 0.1%/本 |
| `triMinTouches` | 2     |
| `triMinBars`    | 15    |
| `triMaxBars`    | 60    |

## マーカー

- ブレイク足に `arrowUp`、略号 `A-Tri`

## 注意

- だまし（一旦抜けて反転）が多い形。MVP は仮検出→N 本維持で確定の 2 段階を検討
